# Chat Feature

Have interactive conversations with AI about your emails.

## Accessing Chat

1. Open an email in Thunderbird
2. Right-click → **Quill** → **Chat**
3. A chat window opens with the email context

## How It Works

The chat feature:
- Loads the current email as context
- Allows multi-turn conversations
- Remembers conversation history
- Uses your configured AI provider

## Use Cases

### Analyzing Emails
```
You: What are the key action items in this email?
AI: Based on the email, here are the action items...

You: When is the deadline mentioned?
AI: The deadline mentioned is...
```

### Drafting Responses
```
You: Help me write a polite decline to this invitation
AI: Here's a suggested response...

You: Make it shorter and more direct
AI: Here's a more concise version...
```

### Understanding Context
```
You: Explain the technical terms in this email
AI: Here are the key terms explained...
```

## Tips

- **Be conversational**: Ask follow-up questions
- **Provide context**: Mention what you need
- **Iterate**: Refine responses through dialogue

## Limitations

- Chat history clears when window closes
- Context limited to current email
- Token limits apply per message
