# Contributing

Help improve Quill! We welcome contributions of all kinds.

## Ways to Contribute

### Report Bugs
1. Check [existing issues](https://github.com/mikecastrodemaria/Quill/issues)
2. Create a new issue with:
   - Thunderbird version
   - Quill version
   - Steps to reproduce
   - Expected vs actual behavior

### Suggest Features
Open an issue with:
- Clear description of the feature
- Use case / why it's useful
- Any implementation ideas

### Submit Code

1. Fork the repository
2. Create a branch: `git checkout -b feature/my-feature`
3. Make your changes
4. Test thoroughly
5. Submit a pull request

## Development Setup

```bash
git clone https://github.com/mikecastrodemaria/Quill.git
cd Quill/plugin
```

Load in Thunderbird for testing - see [[Building]]

## Code Style

- Use clear, descriptive variable names
- Comment complex logic
- Follow existing code patterns
- Test with all three providers

## Pull Request Guidelines

- One feature/fix per PR
- Clear description of changes
- Update documentation if needed
- Test on Thunderbird 78+

## Translation

Help translate Quill:
1. Copy an existing locale file
2. Translate strings
3. Submit PR

Currently available: English, French
