# Basic Usage

Learn how to use Quill's core features for email processing.

## Quick Actions

When composing or reading an email, right-click to access Quill actions:

### Built-in Actions

| Action | Description |
|--------|-------------|
| **Summarize** | Get a concise summary of the email |
| **Translate to English** | Translate email content |
| **Translate to French** | Translate email content |
| **Fix Grammar** | Correct spelling and grammar |
| **Professional Tone** | Rewrite in professional style |
| **Casual Tone** | Rewrite in casual style |
| **Draft Reply** | Generate a response |

## Using Actions

### On Received Emails

1. Open an email
2. Right-click on the email body
3. Select **Quill** → Choose action
4. View result in popup

### On Drafts

1. Compose a new email or reply
2. Write your draft text
3. Right-click → **Quill** → Choose action
4. Result replaces your text (or opens in popup)

## The Draft Window

When you use "Draft Reply" or similar actions:

1. A new window opens with the AI response
2. Review and edit the text
3. Click **Insert** to add to your email
4. Or **Copy** to clipboard

## Tips

- **Be specific**: Select specific text for focused actions
- **Iterate**: Run multiple actions to refine results
- **Custom actions**: Create your own prompts in settings
