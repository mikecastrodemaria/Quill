Configuration
  1. Choisissez votre provider IA (Anthropic, OpenAI ou Ollama local)
  2. Pour Anthropic/OpenAI : obtenez une clé API : Anthropic | OpenAI
  3. Pour Ollama : installez Ollama et téléchargez un modèle (ex: ollama pull llama3.2)
  4. Entrez votre clé API ou URL Ollama ci-dessous
  5. Choisissez votre modèle préféré
  6. N'oubliez pas de cliquer sur Sauvegarder !
Utilisation
  1. Ouvrez une fenêtre de composition (Nouveau, Répondre ou Transférer)
  2. Sélectionnez du texte à traiter
  3. Cliquez sur l'icône Quill (en haut à droite) et choisissez une action
  4. Patientez pendant que l'IA traite votre demande
  5. (Optionnel) Cliquez sur Régénérer pour une nouvelle réponse
